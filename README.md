# Landing Page

This project is included in The Odin Project's curriculum. This project's objective is to use HTML and CSS to create a static landing page while implementing basic layout and stylistic principles.

## Project Overview

A given design layout is intended to be replicated on the landing page. It emphasizes on using HTML to properly organize content and CSS to style it, especially using Flexbox to make responsive layouts.

This project strengthened my comprehension of:

* The construction of HTML pages
* CSS styling
* Flexbox
* Arranging the sections of a webpage

## Features

* Navigation links in the header section
* A hero section with a call-to-action button
* An information cards section
* A quote or testimonial section
* A call-to-action banner
* A footer section

## Built With

* HTML5
* CSS3
* Flexbox
